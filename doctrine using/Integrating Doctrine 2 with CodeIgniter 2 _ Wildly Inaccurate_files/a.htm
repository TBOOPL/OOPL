                            wildlyinaccurate Mod Robert 3 years ago

                            Hi Robert,

                            You should try something like:

                            $this->session->set_userdata('my_object', serialize($object));
                            $object = unserialize($this->session->userdata('my_object'));

                            It is usually not a good idea to serialize Doctrine objects as they can be very big (from all of the relationships). When I need to serialize an object I implement the Serializable interface, like this:

                            class User implements Serializable {
                                protected $id;
                                protected $username;
                                protected $password;

                                public function serialize()
                                {
                                    return serialize(array(
                                        'id' => $this->getId(),
                                        'username' => $this->getUsername(),
                                        'password' => $this->getPassword()
                                    ));
                                }

                                public function unserialize($data)
                                {
                                    $this->id = $data['id'];
                                    $this->username = $data['username'];
                                    $this->password = $data['password'];
                                }
                            }
